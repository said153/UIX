Nombre de la aplicación:
UIX 

Descripción de la aplicación:
Esta aplicación móvil para Android muestra ejemplos básicos de distintos componentes de interfaz de usuario. 
Sirve como guía práctica para entender el funcionamiento de elementos como botones, campos de texto, listas 
y más, dentro de un proyecto Android. El objetivo principal es ayudar a estudiantes y desarrolladores 
principiantes a familiarizarse con la estructura y uso de Activities, Fragments y layouts XML en Android Studio.


Instrucciones de uso:
->Descargar o clonar el repositorio en Android Studio.
->Abrir el proyecto desde Android Studio y esperar a que finalice la sincronización de Gradle.
->Conectar un dispositivo Android o configurar un emulador.
->Ejecutar la aplicación desde Android Studio con el botón Run.
->Una vez instalada, la app mostrará un menú principal desde donde se pueden explorar los diferentes fragments con ejemplos de:
	Campos de texto (EditText)
	Botones (Button, ImageButton)
	Elementos de selección (CheckBox, RadioButton, Switch)
	Listas (RecyclerView / ListView)
	Elementos de información (TextView, ImageView, ProgressBar)
